Mobile Testing Instructions

How to Test Forms & File Upload Mobile Features:

  1. Form Inputs: Test touch interactions, focus states, and keyboard behavior
  2. File Upload: Try drag-and-drop on desktop, tap to upload on mobile
  3. Table Scrolling: Horizontal scroll to see all columns, test input fields
  4. Responsive Layout: Watch forms adapt from multi-column to single-column
  5. Touch Targets: Verify all buttons and inputs are easily tappable (44px+)
  6. Keyboard Navigation: Test tab navigation and form submission
