SMTP Configuration

Configure email server settings for parent notifications

Before You Start

To send emails to parents, you need to configure SMTP settings. Here's what you'll need:

  • SMTP server address (e.g., smtp.gmail.com)
  • SMTP port number (usually 587 for TLS or 465 for SSL)
  • Email account username and password
  • For Gmail: You'll need to use an "App Password" instead of your regular password
Important Security Note

This form generates environment variables that you need to set manually. Never store email passwords directly in your code or database.

For Gmail, use an App Password instead of your regular password
