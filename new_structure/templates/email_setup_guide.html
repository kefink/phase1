Email Setup Guide

Complete guide to configure parent email notifications

Gmail Setup (Recommended)

  1. 1
    Enable 2-Factor Authentication
    Go to your Google Account settings and enable 2-factor authentication. This is required to generate app passwords.
  2. 2
    Generate App Password
    In your Google Account settings, go to Security → App passwords → Generate new app password for "Mail".
    Important

    Use the generated app password, NOT your regular Gmail password!

  3. 3
    Configure SMTP Settings
    Use these settings for Gmail:
    SMTP Server: smtp.gmail.com SMTP Port: 587 Username: your-email@gmail.com Password: [Your App Password] Use TLS: Yes

Setting Environment Variables

Why Environment Variables?

Environment variables keep your email credentials secure and separate from your code.

  1. 1
    Create .env File (Recommended)
    Create a file named .env in your project root directory:
    # Email Configuration SMTP_SERVER=smtp.gmail.com SMTP_PORT=587 SMTP_USERNAME=your-school-email@gmail.com SMTP_PASSWORD=your-app-password-here SMTP_USE_TLS=true BASE_URL=http://localhost:5000
  2. 2
    Alternative: System Environment Variables
    You can also set these as system environment variables:
    # Windows (Command Prompt) set SMTP_SERVER=smtp.gmail.com set SMTP_PORT=587 set SMTP_USERNAME=your-email@gmail.com set SMTP_PASSWORD=your-app-password # Windows (PowerShell) $env:SMTP_SERVER="smtp.gmail.com" $env:SMTP_PORT="587" # Linux/Mac export SMTP_SERVER=smtp.gmail.com export SMTP_PORT=587
  3. 3
    Restart Application
    After setting environment variables, restart your application for changes to take effect.

Testing Your Configuration

  1. 1
    Use the Configuration Form
    Go to Email Configuration → SMTP Settings and enter your details. The system will generate the environment variables for you.
  2. 2
    Send Test Email
    Use the "Send Test Email" button in the Email Configuration dashboard to verify your settings work.
  3. 3
    Check Email Logs
    Monitor the email logs to see if emails are being sent successfully.

Troubleshooting

!
Authentication Failed
Make sure you're using an App Password for Gmail, not your regular password.
!
Connection Timeout
Check your firewall settings and ensure port 587 is not blocked.
!
Environment Variables Not Working
Restart your application after setting environment variables. Check that the .env file is in the correct location.
